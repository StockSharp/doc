# Создание источника

Для добавления собственного источника необходимо создать собственный [IMessageAdapter](xref:StockSharp.Messages.IMessageAdapter). Как это сделать описано в пункте [Создание собственного коннектора](../api/connectors/creating_own_connector.md). После создания необходимо добавить получившийся dll файл в папку с [Hydra](../hydra.md).
