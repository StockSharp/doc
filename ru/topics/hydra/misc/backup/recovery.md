# Восстановление данных

Вы можете восстановить данные из хранилища двумя способами:

- Загрузить сохраненные объекты из консоли AWS. Для этого нужно щелкнуть правой кнопкой по объекту и выбрать **Download** из контекстного меню.![Aws3 download](../../../../images/aws3_download.png)

  > [!TIP]
  > Данный способ позволяет загружать только по одному объекту (файлу) за раз.
- Использовать бесплатную программу для загрузки данных [S3 Browser](https://s3browser.com/).
