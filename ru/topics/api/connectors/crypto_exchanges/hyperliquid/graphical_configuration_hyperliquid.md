# Графическое конфигурирование Hyperliquid

Для всех продуктов [S\#](../../../../api.md) графическая настройка подключения выполняется в форме [Окно настройки подключений](../../../graphical_user_interface/connection_settings_window.md).

Основные поля:

- **Wallet address** - адрес кошелька (`0x...`).
- **Private key** - приватный ключ для подписанных действий.
- **Vault address** - опциональный адрес vault/sub-account.
- **Expires after** - опциональная метка времени истечения подписи.
- **Section** - торговая секция (`Spot` или `Derivatives`).
- **Testnet** - тестовые endpoint адреса.
- **Info endpoint / Exchange endpoint / WS endpoint** - endpoint адреса.
- **Market slippage** - значение slippage по умолчанию для рыночных заявок.
