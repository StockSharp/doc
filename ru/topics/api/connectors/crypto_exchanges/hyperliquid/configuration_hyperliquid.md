# Настройки коннектора Hyperliquid

Для работы с коннектором необходимо указать данные кошелька для торговых действий.

Основные настройки:

- **Wallet address**.
- **Private key**.
- **Vault address** (опционально).
- **Expires after** (опционально).
- **Section**: `Spot` или `Derivatives`.
- режим **Testnet**.
- **Info endpoint / Exchange endpoint / WS endpoint**.
- **Market slippage** для рыночных заявок.

Официальная документация API:

- [Hyperliquid API](https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api)
- [Info endpoint](https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api/info-endpoint)
- [Exchange endpoint](https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api/exchange-endpoint)
- [Websocket](https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api/websocket)
