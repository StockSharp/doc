# ITCH

**ITCH** \- сервис поставки маркет\-данных в режиме реального времени котировок акций, Форекс, фьючерсных контрактов и др.

До того, как приступить к написанию торговых роботов под текущую торговую платформу, рекомендуется ознакомиться с ссылками, приведенными в пункте [Коннекторы](../../connectors.md). 

> [!TIP]
> [ITCH]() используется только для получения биржевых данных, торговые транзакции не поддерживаются.

В [S\#](../../../api.md) для работы с [ITCH]() необходимо использовать [ItchMessageAdapter](xref:StockSharp.ITCH.ItchMessageAdapter).
