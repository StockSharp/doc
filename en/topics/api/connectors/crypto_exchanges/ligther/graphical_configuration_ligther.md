# Graphical configuration Ligther

For all [S\#](../../../../api.md) products, graphical configuration is performed in the [Connection settings window](../../../graphical_user_interface/connection_settings_window.md).

Main fields:

- **Key** - API key.
- **Secret** - API secret.
- **Account index** - account index.
- **API key index** - API key index.
- **Section** - trading section (`Spot` or `Derivatives`).
- **Demo** - demo environment mode.
- **Spot REST / Derivatives REST** - REST endpoints.
- **Spot WS / Derivatives WS** - websocket endpoints.
- **WS read-only mode** - websocket read-only mode flag.
