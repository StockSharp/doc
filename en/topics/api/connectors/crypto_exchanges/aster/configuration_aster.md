# Configuration Aster

To work with the connector, generate **API Key** and **Secret** in the exchange account and specify them in connection settings.

Main settings:

- **Key** and **Secret**.
- **Section**: `Spot` or `Derivatives`.
- **Derivatives mode**: `Legacy` or `V3 Agent`.
- **Spot REST / Spot WS** endpoints.
- **Derivatives REST / Derivatives WS** endpoints.
- **Demo** mode.

> [!TIP]
> Aster derivatives have two protocol families. Select the correct **Derivatives mode** before enabling trading.
