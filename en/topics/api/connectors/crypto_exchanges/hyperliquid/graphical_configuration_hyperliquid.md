# Graphical configuration Hyperliquid

For all [S\#](../../../../api.md) products, graphical configuration is performed in the [Connection settings window](../../../graphical_user_interface/connection_settings_window.md).

Main fields:

- **Wallet address** - wallet address (`0x...`).
- **Private key** - private key for signed actions.
- **Vault address** - optional vault or sub-account address.
- **Expires after** - optional signature expiration timestamp.
- **Section** - trading section (`Spot` or `Derivatives`).
- **Testnet** - testnet endpoints.
- **Info endpoint / Exchange endpoint / WS endpoint** - endpoint URLs.
- **Market slippage** - default slippage for market orders.
