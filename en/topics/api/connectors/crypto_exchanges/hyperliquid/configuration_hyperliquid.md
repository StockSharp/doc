# Configuration Hyperliquid

To work with the connector, specify wallet-level trading credentials.

Main settings:

- **Wallet address**.
- **Private key**.
- **Vault address** (optional).
- **Expires after** (optional).
- **Section**: `Spot` or `Derivatives`.
- **Testnet** mode.
- **Info endpoint / Exchange endpoint / WS endpoint**.
- **Market slippage** for market orders.

Official API documentation:

- [Hyperliquid API](https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api)
